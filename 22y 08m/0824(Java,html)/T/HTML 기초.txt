- html의 기본 양식
<!DOCTYPE html>
<html>
<head>		</head>		<!--head: 탭부분 디자인-->
<body>		</body>		<!--body: 페이지 디자인-->
</html>
<!--중괄호 열고 닫는 것처럼 태그도 열고 닫아야함-->

- h 태그
<h1>, <h2>, <h3>, <h4>, <h5>, <h6>
(h1이 가장 글자가 크고, h6이 가장 작음)

- 속성: 태그를 더 상세히 표현
		href: 하이퍼링크 속성
(img 태그)	src: 어떠한 사진 넣는지 명시하는 역할의 속성
			현재 기기의 사진 이름이나 이미지 링크 삽입하여 설정
		width와 height 통해 사진 크기 설정
		alt: 이미지 가져오지 못할 때, 대체 텍스트 지정
